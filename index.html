Brief Description

MyFestival is an online Festival guide where users could add, delete or modify existing information about a Festival. Then the information will be sent to our Android mobile app.

The MVC app will be on a web host soon.

This project took place over a 7 month period, and once every week I meet with my project supervisor. My project specification was:
  • Web app built using MVC 4. Using Partial Views, ViewModels, Controllers, Views.
  • Styled with Twitter Bootstrap 3.0., JQuery, JavaScript, Bootstrap Datepicker.
  • Database : LocalDb/SqlAzure optimization
  • Mobile app: Xamarin.Android with C#.
  • WCF Service App: To pull informartion out from the database and bring data down into the app.
  • Managing version control
I Achieved:
  • Got the MVC appplication working perfectly. Was hoping to add Geolocation/Google Maps input but ran out of time.
  • Database : Working up on SqlAzure.
  • Mobile app: Xamarin.Android with C#. Got some of the information to be pulled down.
  • WCF Service App: Got this to work fully
  • Managing version control
